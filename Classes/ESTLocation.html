Inherits from NSObject

ESTLocation.m

Overview

Represents a physical location prepared for Estimote Indoor Location. Object is immutable.

Instances of ESTLocation can be created by ESTLocationBuilder or by importing JSON with [ESTLocationBuilder parseFromJSON:].
